## ⚠️ 已知的遗留项/限制

### 1. CVE.org 批量获取 (设计限制)
```python
# cve_analyzer/fetcher/cve_org.py
def fetch(self, since=None, until=None) -> List[CVE]:
    # CVE.org 不提供批量查询接口
    return []
```
**影响**: 无法通过 CVE.org 批量获取  
**解决**: 仅用于单个 CVE 补充信息

### 2. 增量同步优化 (可改进)
- **当前**: 全量获取指定时间段
- **问题**: 重复抓取已存在的 CVE
- **优化**: 基于 `lastModified` 只获取更新的

### 3. 抓取进度显示 (体验优化)
- **当前**: 只有开始和完成提示
- **缺失**: 实时进度条、预计剩余时间
- **影响**: 大范围抓取时用户不知道进度

### 4. 数据源独立控制 (功能缺失)
```bash
# 当前 --source 参数未完全实现
# 协调器总是使用所有启用的 fetcher
```

### 5. 抓取中断恢复 (功能缺失)
- **当前**: 中断后需重新抓取
- **缺失**: 断点续传、已抓取记录

### 6. 高级过滤 (功能缺失)
- **缺失**: 按严重程度过滤抓取
- **缺失**: 按厂商/产品过滤
- **缺失**: 按 CVSS 分数过滤
