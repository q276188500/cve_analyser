# CVE Review SKILL

CVE 漏洞审查与影响评估 Agent，基于 OpenCLAW 框架。

## 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                     CVE Review SKILL                       │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐   │
│  │   输入层    │───▶│  分析引擎   │───▶│   输出层    │   │
│  │ 时间段/CVE  │    │ OpenCLAW   │    │ Markdown    │   │
│  │ 代码仓路径  │    │ (内核专家) │    │ 报告       │   │
│  └─────────────┘    └──────┬──────┘    └─────────────┘   │
│                             │                              │
│         ┌───────────────────┼───────────────────┐          │
│         ▼                   ▼                   ▼          │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐   │
│  │ cve-analyzer│    │  知识库    │    │  代码仓    │   │
│  │   (工具)    │    │ (规则)     │    │ (内核源码) │   │
│  └─────────────┘    └─────────────┘    └─────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

## 核心流程

### Step 1: 数据获取

- 从 cve-analyzer 同步 CVE 数据
- 支持时间段或 CVE ID 查询

### Step 2: 代码仓检查（新增）

- 读取 `config/config.yaml` 获取代码仓路径
- **如果不存在**，提示用户获取
- 每次分析前同步最新代码

### Step 3: 初步筛选

基于 `.config` 配置：

| 条件 | 结果 |
|------|------|
| KCONFIG 未开启 | defer |
| 非内核问题 | defer |

### Step 4: 详细评估

由 **OpenCLAW（内核专家）** 主导：
1. CVE 详情获取
2. Patch 代码分析
3. 代码仓代码查询
4. 知识库匹配
5. 综合判断

### Step 5-7: 报告生成与归档

- 输出 Markdown 报告
- 归档到 `reports/年份/月份/`

## 使用方法

```bash
# 分析指定时间段 CVE
分析 2025年11月 CVE

# 分析具体 CVE
CVE-2025-40107
```

## 报告示例

```markdown
# CVE 分析报告: CVE-2025-40107

## 基本信息
| CVE ID | 严重程度 | 披露日期 |
|--------|----------|----------|
| CVE-2025-40107 | UNKNOWN | 2025-11-03 |

## 漏洞类型
空指针解引用 (Null Pointer Dereference)

## 受影响文件
- drivers/net/can/spi/hi311x.c

## Kconfig 检查
| 配置 | 状态 |
|------|------|
| CONFIG_CAN | 未设置 |

## 合入建议
**动作**: defer
**理由**: KCONFIG未开启，无实际影响

## 内核专家详细分析
### Patch 分析
[代码分析...]

### 影响分析
[详细影响...]

---
*分析时间: 2026-03-19*
```

## SKILL 规范

符合 OpenCLAW skill-creator 规范：
- ✅ YAML frontmatter (name + description)
- ✅ SKILL.md 定义完整流程
- ✅ scripts/ 集成 cve-analyzer
- ✅ knowledge/ 知识库
- ✅ config/ 配置文件
- ✅ reports/ 报告归档
